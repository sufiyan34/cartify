Cartify – Mini E-Commerce App
Tagline: Shop Smart. Shop Easy.
Cartify is a fully functional mini e-commerce mobile app built with Flutter and GetX, simulating a complete online shopping experience end-to-end using local mock data — no backend required. It demonstrates clean state management, reactive UI, named routing, and a polished Material 3 design across 13 screens.
Key Features

Authentication – mock login/signup with a single toggleable form
Product Catalog – 14 products across 7 categories, with category filtering, search, and sort (price low/high, top rated)
Product Details – image carousel, ratings/reviews, stock status, quantity selector, related products
Cart – add/remove items, quantity controls, live subtotal/tax/shipping/total calculation
Wishlist – heart-toggle favorites, synced reactively across all product views
Checkout – multiple saved addresses (add new via dialog), payment method selection, itemized order summary
Order Management – confirmation screen, order history, and detailed order view with a visual status tracker (Placed → Processing → Shipped → Delivered)
Profile – user info, saved addresses, settings shortcuts, logout

Tech Stack

Flutter (Material 3)
GetX – state management, dependency injection, and navigation
intl – date formatting
Local mock data layer (no API dependency)
